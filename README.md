# 🚀 Customer Churn Prediction using CatBoost

## 📌 Overview
Customer churn is one of the biggest challenges for businesses like telecom, SaaS, and banking.

This project focuses on predicting whether a customer will leave (churn) or stay using Machine Learning.

Using **CatBoost**, we build a powerful classification model that handles categorical data efficiently and delivers strong performance.

---

## 🎯 Objectives
- Analyze customer behavior using EDA
- Identify key factors affecting churn
- Build a predictive ML model
- Evaluate model performance

---

## 📊 Exploratory Data Analysis (EDA)
Performed advanced EDA to uncover insights:

- Distribution plots (Histogram + KDE)
- Countplots for churn imbalance
- Boxplots & Violin plots for feature comparison
- Scatterplots for relationship analysis
- Correlation heatmap
- Pairplot for multi-feature visualization

---

## 💡 Key Insights
- Customers with lower tenure are more likely to churn
- Higher monthly charges increase churn probability
- Contract type significantly impacts retention
- Certain services like tech support reduce churn
